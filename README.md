# Laravel Library System

## Installation

Install my-project with npm

```bash
  composer install
```
```bash
  cp .env.example.com .env 
```
```bash
  php artisan key:generate
```
```bash
  npm install && npm run dev
```
```bash
  php artisan migrate
```
```bash
  php artisan db:seed
```
```bash
  php artisan serve
```
